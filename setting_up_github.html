Overview

Git jargon: https://help.github.com/articles/github-glossary/

Basics steps:

  • clone: copy the remote code repository locally

  • pull: fetch and merge remote changes

  • commit: “save” and document your modifications locally

  • push: “export” your modifications to the remote repository
